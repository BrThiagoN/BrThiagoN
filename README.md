### 🚀 Projetos em Destaque

*   🎸 **[music-store-cli](https://github.com/BrThiagoN/music-store-cli)** — *Arquitetura de Software & CLI*
    *   Aplicação de terminal em **Python** construída sobre sólidos princípios de **Programação Orientada a Objetos (POO)**. Implementa um fluxo robusto de gerenciamento de inventário (CRUD), persistência e validações rigorosas de dados para simular cenários reais de gestão comercial.
*   🍷 **[vinharia-agnelo-arduino](https://github.com/BrThiagoN/vinharia-agnelo-arduino)** — *IoT & Sistemas Embarcados*
    *   Solução inteligente em **C++** integrada a hardware físico (Arduino, LDR e DHT11) para preservação ambiental de vinhos. O sistema realiza leitura e análise em tempo real, disparando alertas sonoros e visuais no display LCD quando limites térmicos são violados.
*   🐾 **[Landing-Page---Pet-Shop-Patas-Pelos](https://github.com/BrThiagoN/Landing-Page---Pet-Shop-Patas-Pelos)** — *UI/UX & Conversão*
    *   Landing page responsiva de alta conversão utilizando **HTML5, CSS3 e JavaScript**. O projeto traz animações fluidas, modo escuro nativo e uma calculadora de serviços interativa para simulação instantânea de orçamentos pelo cliente.
*   📖 **[projetoTradutorDeTermoTecnico](https://github.com/BrThiagoN/projetoTradutorDeTermoTecnico)** — *Acessibilidade & Educação*
    *   Interface web leve criada para democratizar o ecossistema de TI para iniciantes, traduzindo conceitos e termos técnicos complexos para uma linguagem simples e didática.
